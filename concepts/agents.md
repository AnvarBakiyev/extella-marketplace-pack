# concept: Готовые агенты-роли (14)

Готовые бизнес-роли для платформенного агента Qwen. Каждая роль — системная инструкция (в `agents/<id>.md`) + протокол сервисов. Роль «прошивается» в UI-копию агента через эксперт `agent_flash_role(agent_id, role_id)` (agent/update; provider/model не трогаются — остаётся Qwen), и при этом в скоуп агента провижинятся сервисы `svc_*`.

Категории и роли:
- **Продажи** — `sdr` (Менеджер по продажам), `account` (Аккаунт-менеджер).
- **Маркетинг** — `smm` (SMM-контент), `copywriter` (Копирайтер).
- **Поддержка** — `support` (1-я линия), `faq` (FAQ-бот).
- **Финансы** — `bookkeeper` (Помощник бухгалтера), `finanalyst` (Финансовый аналитик).
- **HR** — `recruiter` (Рекрутер), `onboarding` (Онбординг).
- **Операции** — `assistant` (Бизнес-ассистент), `docs` (Документы), `pm` (Ассистент PM), `analyst` (Бизнес-аналитик).

Как создать: пользователь делает UI-копию базового агента в Extella (2 клика), берёт её ID, затем `agent_flash_role(agent_id, role_id)` прошивает инструкцию роли + подключает сервисы. Полные инструкции — в папке `agents/`.
