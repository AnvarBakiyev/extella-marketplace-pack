# Агенты-роли (14)


Каждый файл — готовая бизнес-роль: системная инструкция агента (модель Qwen) + протокол сервисов.

Файлы `agents/<role_id>.md` — единственный редактируемый источник инструкций. После изменения роли запусти `python3 tools/build_agent_flash_role.py`; команда пересоберёт компактный payload эксперта `agent_flash_role`. Режим `--check` и тесты релиза проверяют точное совпадение всех 14 ролей и ограничение размера облачного эксперта.
